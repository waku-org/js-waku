    js-waku

    A TypeScript implementation of the Waku v2 protocol.

    Documentation

    • Quick start
    • Full documentation
    • API documentation (master branch)
    • Waku
    • Vac

    API Documentation can also be generated locally:

    git clone https://github.com/waku-org/js-waku.git
    cd js-waku
    npm install
    npm run doc

    Using Nix shell

    git clone https://github.com/waku-org/js-waku.git
    cd js-waku
    nix develop
    npm install
    npm run doc
